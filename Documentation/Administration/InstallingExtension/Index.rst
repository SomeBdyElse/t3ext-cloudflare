Installing the extension
^^^^^^^^^^^^^^^^^^^^^^^^

There are a few steps necessary to install the CloudFlare Client
extension. If you have installed other extensions in the past, you
will run into little new here.


Install the extension from Extension Manager
""""""""""""""""""""""""""""""""""""""""""""

The CloudFlare Client extension can ben installed through the typical
TYPO3 installation process using the Extension Manager.
